Paste your Google Sheets sharing URL below, then generate a QR code to print or save.

How to set this up:
  1. Open your Excel file, go to File → Save As → Google Sheets (or upload at drive.google.com)
  2. In Google Sheets, click Share → Change to anyone with the link → Viewer
  3. Copy the link and paste it above
  4. Generate and download the QR code
  5. Print and laminate it — stick it inside the boat near the harness
